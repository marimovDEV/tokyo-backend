📌 BIRINCHI QADAMLAR:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. SSH orqali Beget serverga ulanish:
   ssh u1234567@tokyokafe.uz

2. Loyiha papkasiga o'tish:
   cd ~/public_html/backend

3. Barcha skriptlarda BEGET_USER ni o'zgartirish:
   
   Ochish:           nano start_beget.sh
   O'zgartirish:     BEGET_USER="u1234567"  ← O'zingiznikini yozing
   Saqlash:          Ctrl+O, Enter, Ctrl+X
   
   Boshqa skriptlar ham shunday:
   - stop_beget.sh
   - status_beget.sh  
   - check_and_restart.sh

4. Virtual environment yaratish (agar yo'q bo'lsa):
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

5. Django sozlash:
   python manage.py migrate
   python manage.py collectstatic --noinput

❌ Server ishga tushmayapti?
   → tail -50 logs/gunicorn_error.log

❌ Port band?
   → netstat -tuln | grep 8000

❌ Eski jarayonlar?
   → ps aux | grep gunicorn
   → kill -9 <PID>

❌ Virtual environment aktivmasmi?
   → source venv/bin/activate
   → which python

❌ Permission denied?
   → chmod +x *.sh
   → chmod -R 755 logs/
